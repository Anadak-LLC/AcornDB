- **Batched (Azure Table):** Up to 100 items per request

### Read Performance
- **Memory:** <1ms (cache hit)

- **RDBMS:** 5-20ms (indexed)
- **Cloud:** 50-200ms (network latency)

### IRoot Pipeline Overhead
- **Compression (Brotli):** ~10-30% overhead, 50-80% size reduction
- **Encryption (AES):** ~5-15% overhead
- **Policy:** <5% overhead
- **Combined:** ~20-50% overhead (varies by data)
